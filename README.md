# 📰 AI-Powered Fake News Detection

## 📌 Overview

This project is an AI-powered Fake News Detection system that classifies news articles as Real or Fake using Natural Language Processing (NLP) and Machine Learning techniques.

The application provides an interactive Streamlit web interface where users can paste a news article and instantly receive a prediction along with the model confidence score.

---

## 🚀 Features

* News Classification (Real vs Fake)
* NLP Text Preprocessing
* Stopword Removal
* Stemming
* TF-IDF Vectorization
* Multiple Machine Learning Models
* Interactive Streamlit Web Application
* Confidence Score Display

## 📈 Results

| Model               | Accuracy |
| ------------------- | -------- |
| Logistic Regression | 98%+     |
| Naive Bayes         | 92%+     |
| Random Forest       | 99%+     |
| SVM                 | 99%+     |

Random Forest achieved the best performance and was selected for deployment.
